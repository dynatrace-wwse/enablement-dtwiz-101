# DTWiz 101 - Image assets

The `*.png` files in this folder are **placeholders** (currently a reused
framework image) so the MkDocs site and the Enablement App render without broken
images. Replace each with a funny, on-theme illustration before publishing.

| File | Intended illustration | Used on |
| --- | --- | --- |
| `dtwiz-wizard.png` | DTWiz the wizard mascot, confident and slightly chaotic | `index.md` |
| `dtwiz-conjure.png` | Wizard conjuring a Kubernetes cluster (k8s wheel) out of thin air | `01-install-kubernetes.md` |
| `dtwiz-todo.png` | An over-serious "enterprise" Todo app / quest board | `02-deploy-todo-app.md` |
| `dtwiz-eye.png` | The all-seeing Dynatrace eye watching the app | `03-observe-with-dynatrace.md` |
| `dtwiz-chaos.png` | Wizard casting a chaos spell and immediately regretting it | `04-troubleshoot-and-recommend.md` |

Guidance:
- Keep them light and humorous but professional (this ships in the Dynatrace app).
- Square-ish, ~600x600, transparent PNG works best with the `align=right` layout.
- Source from an approved image library or generate; do not use unlicensed art.
